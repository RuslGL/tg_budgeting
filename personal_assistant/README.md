# Personal Assistant Bot

Бот для ведения CRM по друзьям и знакомым. Данные хранятся в Google Sheets.

## Таблица

Spreadsheet ID: задаётся в `.env` как `ASSISTANT_SPREADSHEET_ID`
Лист: `friends`

| Колонка | Название | Описание |
|---------|----------|----------|
| A | Имя | Цвет = статус отношений |
| B | ДР | День рождения |
| C | Комментарий | Заметки с датой: "18.04: ..." |
| D | Запланировано | Описание события. Цвет = тип встречи |
| E | Дата события | Дедлайн: "до 26.04" или "26.04.2026" |
| F | Итог | Результат с датой: "18.04: встретились" |

### Цвета колонки A (статус контакта)
- Красный — нет контакта, искать
- Жёлтый — был контакт, но увял
- Зелёный — есть периодический контакт

### Цвета колонки D (тип встречи)
- Голубой (#ADD8E6) — живая встреча
- Фуксия (#FF77FF) — онлайн

## Команды (голос или текст)

### Просмотр
- `/contacts` или "список контактов" — показать всех
- `/columns` или "что куда записывать" — описание колонок
- "покажи кто на этой неделе" — дедлайны на 7 дней
- "покажи всех без плана" — контакты без события

### Обновление данных
- "перенеси Женю Кизилова на 5 мая" — обновить дату события
- "выполнено Женя, следующая встреча до 1 июня" — записать итог, обновить дату
- "запиши в комментарии к Роману: предпочитает вечера" — добавить заметку
- "запиши ДР Романа 15 марта" — день рождения
- "отмени встречу с Женей" — очистить событие и дату

### Цвета
- "встретились с Женей, живая встреча" — покрасить D в голубой
- "онлайн с Романом" — покрасить D в фуксия
- "статус Женя — периодический контакт" — покрасить A в зелёный
- "статус Женя — красный" — покрасить A в красный

## Напоминания

- За 7→1 день до дедлайна: каждый день в 8:00
- В день дедлайна: в 8:00, 12:00, 18:00, 21:00
- Просрочено: каждый день в 8:00
- Каждую пятницу в 8:00: список контактов без запланированных событий
