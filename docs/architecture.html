Жизненный цикл сообщения

Транзакция (бот семья / бизнес)

User → отправляет текст или голос
  │
  ├─ голос → transcription.py → Whisper API → текст
  │
  └─ текст → llm.py → gpt-4o-mini
               │
               ├─ amount + category найдены
               │   └─ sheets.py → raw (Google Sheets)
               │       └─ "Внесено: дата сумма категория"
               │
               ├─ amount не найден → FSM clarifying_amount
               │   └─ бот: "Укажи сумму"
               │       └─ ответ → повтор (макс 3 попытки)
               │
               └─ category не найден → FSM clarifying_category
                   └─ бот: кнопки со всеми категориями
                       └─ выбор → sheets.py → raw

dashboard sync (каждый час):
  Google Sheets raw → SQLite transactions
  Browser /d/secret → React App → charts

Заметка (bot_notes)

User → отправляет текст или голос
  │
  ├─ голос → transcription.py → Whisper API → текст
  │
  └─ текст → llm.py → parse_note()
               │         возвращает {category, event_date}
               │
               ├─ category найдена, не Календарь
               │   └─ sheets.py → notes (Google Sheets)
               │       └─ "Записано: дата · категория"
               │
               ├─ category = Календарь, event_date найдена
               │   └─ sheets.py → notes + Google Calendar
               │       └─ "Записано: дата · Календарь\nСобытие: event_date"
               │
               ├─ category = Календарь, event_date не найдена
               │   └─ FSM clarifying_date
               │       └─ бот: "Укажи дату события"
               │           └─ LLM парсит дату → сохраняет
               │
               └─ category = "unknown"
                   └─ FSM clarifying_category
                       └─ бот: кнопки по алфавиту
                           └─ клик → проверяет Календарь
                              → сохраняет + событие если нужно

dashboard sync (каждые 10 секунд):
  Google Sheets notes → SQLite notes
  Frontend авто-поллинг каждые 10с
  Кнопка "Удалить" → DELETE API
    → удаление из Google Calendar (если есть event_id)
    → удаление из Google Sheets
    → удаление из SQLite

Взаимодействие со Sheets

Запись транзакции:
  bot/bot_corp → services/sheets.py
  → raw.append_row(7 колонок)

Запись заметки:
  bot_notes → services/sheets.py
  → notes.append_row(5 колонок)
  → (если Календарь) Google Calendar API

Удаление заметки:
  dashboard → notes_sync.delete_note_from_sheets()
  → Calendar API delete (если event_id)
  → notes.find(uuid) → delete_rows()

Синхронизация в SQLite:
  raw → transactions (каждый час)
  notes → notes (каждые 10 секунд)
